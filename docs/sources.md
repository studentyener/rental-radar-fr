# Sources de données

## DVF — Demandes de Valeurs Foncières
**URL** : https://www.data.gouv.fr/datasets/demandes-de-valeurs-foncieres-geolocalisees
**Licence** : Licence Ouverte 2.0
**Mise à jour** : 2 fois par an (avril et octobre)
**Couverture** : France métropolitaine + DOM (sauf Alsace, Moselle, Mayotte)

## DPE — Diagnostics de Performance Énergétique
**URL** : https://data.ademe.fr/datasets/dpe-v2-logements-existants
**Licence** : Licence Ouverte 2.0

## INSEE
- Population : https://www.insee.fr/fr/statistiques/series/106282028
- Revenus (FiLoSoFi) : https://www.insee.fr/fr/statistiques/7651654

## Zonage Pinel
**URL** : https://www.ecologie.gouv.fr/zones-tendues
Fichier à déposer dans `data/external/zonage_pinel.csv`
Format : `code_commune,zone_pinel` (zones : Abis, A, B1, B2, C)
